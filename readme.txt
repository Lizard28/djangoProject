Открыть консоль
ctrl + ~

Создать виртуальное окружение
python -m venv venv

Вручную активировать виртуальное окружение
venv\scripts\activate.bat

Обновить пакетный менеджер
python -m pip install --upgrade pip

Установить модуль django
pip install django

...

Запустить сервер в консоли
python.exe manage.py runserver 127.0.0.1:8000

Запустить сервер через задачи
ctrl + shift + B
